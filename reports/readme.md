# 🧠 Behavioral User Segmentation for E-Commerce

End-to-end data project that builds a behavioral segmentation pipeline using unsupervised machine learning to identify high-value customer clusters and business growth opportunities.

---

## 🚀 Project Overview

This project analyzes user-level interaction data from an e-commerce environment to:

- Detect behavioral customer segments
- Identify high-value and high-risk clusters
- Enable conversion optimization
- Support CRM & personalization strategies

The pipeline includes:

- Data quality validation
- Feature engineering
- Sessionization
- Content classification
- KMeans clustering
- Business-oriented model selection
- Executive-level reporting

---

## 📊 Key Results

We identified **7 distinct behavioral user segments** across 14,701 users.

### Strategic Insights

- 2 dominant mass segments (65% of users)
- 1 high-value power buyer cluster
- 1 high-conversion low-engagement cluster
- 1 deep-research segment
- 1 category-specialist niche
- 1 statistical micro-segment

The segmentation enables:

- Funnel optimization
- Retargeting strategies
- LTV-based marketing allocation
- Personalization engines
- CRM automation
